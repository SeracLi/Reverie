# First Session 黄金路径 · 蓝月亮 (Blue Moon)

> 草案 B "她不该记得这件事" 的笔友化版
> 用作 First Session 可点切片的脚本基底
> 2026/06/29 · 第二轮

---

## 0. 一句话

**1923 年纽约禁酒令时代, 你被朋友带进一家地下酒吧。当家舞女 Iris 看了你一眼, 走下台。但她身后, 角落里有两个西装男一直在看她。这一个小时, 她做了一个选择。**

---

## 1. 设计目标

这 10 个 beat 必须同时打到三件事:

| 想让看 demo 的人体感到的 | First Session 怎么实现 |
|----|----|
| 1. 这产品和 Tipsy 有什么不同 | Beat 7: Iris 引用你 4 个 beat 前输入的一个细节(#1 字面引用)— 看 demo 的人**亲身体验**到"她记得"|
| 2. 这产品的灵魂层是什么 | Beat 5 / B9: 她的"躲谁"是双关, 她身边有看不见的危险; 她在化妆间里那一瞬的脆弱 = 求而不得引擎的具象化 |
| 3. 为什么这不只是个互动小说 | Beat 10: "我有时会写信……也给从未存在过的人" — 通道开启远期伏笔, 看完一个 session 就知道这关系不止一次 |

---

## 2. 故事设定

| 项 | 内容 |
|---|------|
| **时空** | 1923 年纽约, 禁酒令第 3 年, 11 月下旬, 雨夜 |
| **场景** | 一家叫 **"蓝月亮"(Blue Moon)** 的 Speakeasy(非法地下酒吧)|
| **她** | **Iris** — 当家舞女兼歌手, 22 岁前后, 黑发, 灰蓝色眼睛, 神态独立但藏着倦意 |
| **Iris 的暗线**(first session 完全隐含, 不揭露) | 她是蓝月亮幕后的人 — Dutch Salvatore(一个爱尔兰裔黑帮头目)的"女人" — 蓝月亮上下默认她"属于"Dutch。她不愿意。她攒了三个月私房钱, 在化妆台抽屉里藏着一张去芝加哥的火车票, 但她还没下定决心走 |
| **你** | 玩家第一人称, 来纽约出差的外来者。朋友 Tommy 是常客, 今晚带你来听音乐 |
| **Iris 的"她隐约感到你不属于这里"** | 这是 v1.1 笔友化的核心伏笔, B9 出现, 不揭露 |
| **时长** | 9-10 分钟 |
| **Beat 数** | 10 |
| **触发的"她记住了"** | #1 字面引用(B3 → B7)+ #2 选择回响(B5 → B10)|
| **关键 trait 累积** | romantic / honest / playful / mysterious / brave |

---

## 3. 逐 Beat 脚本

> **格式**:
> - **B[n]** = Beat 编号
> - **[类型]** = 叙事 / 选项 / 自由文本 / 抉择 / 沉浸图 / 触发
> - **时长** = 这个 beat 预期用户停留时间
> - **双轴** = 剧情轴 / 关系轴 权重(0-3)
> - **trait** = 这个 beat 会累积的 trait tag
> - **创作者意图** = 这个 beat 为什么这样设计

### B1 · 雨夜街口 [叙事 + 沉浸图]
**时长**: 30 秒 / **双轴**: 0/0

```
[沉浸图: 1923 年纽约一条无名小巷, 雨水把砖墙浸成深红色,
 一盏红铜油灯挂在不起眼的木门旁, 玻璃罩里火苗在抖。墙上贴着早已被雨打烂的旧海报, 看不出原本写着什么。]

旁白:
1923 年, 11 月。纽约。禁酒令第三年。
你站在一条没有路牌的小巷里, 雨水顺着帽檐滴下来, 落在你皮鞋的扣子上。

前面五步, 朋友 Tommy 弓着身子, 在那扇没有把手的木门上敲 —
三下, 停顿三秒, 再敲两下。
雨声把第一个声音吃掉了。
他重来一遍。

门后传来一只男人的声音, 低哑:
"今晚的暗号是?"

Tommy 用嘴角飞快地说了一句你没听清的话。
门 — 吱呀一声 — 打开了一条缝。
```

**用户动作**: 长按继续 (tap-to-continue)
**创作者意图**: 0 摩擦开场, 立刻给世界观氛围。**雨 / 红铜油灯 / 暗号 / 没有把手的门 / 烂海报** — 五个具体物件定调 noir。比"1920s 纽约"这种抽象描述强 10 倍

### B2 · 进入 [叙事 + 沉浸图] ⚠️ 第一个暗线伏笔
**时长**: 45 秒 / **双轴**: 0/0

```
[沉浸图: 酒吧内部斜俯视图,
 暖橙色钨丝灯 + 几根缓慢燃烧的蜡烛,
 烟雾在灯柱里翻腾, 远处一个三人爵士乐队 (钢琴/贝斯/萨克斯),
 吧台是深胡桃木, 镜面墙上摆着违禁的威士忌, 标签都被刻意撕掉了一半]

旁白:
门后是另一个世界。

一股气味先撞上你 — 威士忌、烟草、女人的香水、湿羊毛大衣、还有一种你说不清的、像是过期的玫瑰的甜味。

灯光是暖橙色的, 三盏蜡烛在角落桌上烧得很慢, 火苗每隔几秒就被人走过时带起的风晃一下。

舞台上, 一个穿黑色礼服的女人正在唱 "My Man" — 是个老歌, 但她唱得像在和一个不在场的人说话。

Tommy 把帽子甩到衣帽间柜台上, 朝吧台走:
"来吧, 给你看点真东西。"

你跟上去。

视角扫过场内 —
四张桌, 八九个客人。
最角落的那张, 坐着两个穿黑色西装的男人。
其中一个的衬衫领口微微敞开, 你看见一截皮带扣下面金属的反光 — 可能是枪柄, 也可能不是。
他们没在喝酒。
他们一直看着舞台。
```

**用户动作**: tap-to-continue
**🌱 暗线伏笔 #1**: 角落两个西装男是 Dutch 的手下。first session 不揭露他们是谁, 但视觉留印
**创作者意图**:
- 五感包裹(气味是最先撞上你的, 不是视觉) — Tipsy 同款写法但要更克制不下流
- **蜡烛每隔几秒被风晃一下** — 这是 Serac 反馈里点名要的那种细节
- 第一个剧情张力暗示: 这不只是一家普通酒吧, 有人在监视

### B3 · 吧台 [自由文本] 🌱 trait 收集起点 + #1 字面引用 setup
**时长**: 1 分钟 / **双轴**: 1/0

```
[沉浸图: 吧台特写,
 一个胖墩墩的酒保侧脸 (50 多岁, 秃顶, 油亮的胡子), 穿白衬衫 + 黑色背带裤, 一条擦得发亮的银表链斜挂在胸前]

旁白:
你坐到吧台前的高脚凳上。
皮面凳子还带着上一个客人的余温。

酒保的眼神先扫过 Tommy — 点了一下头, 像是认得 — 然后落在你身上, 停了大约一秒。

他把抹布甩到肩膀上, 两只胳膊撑着吧台, 朝你倾过来一点。

酒保 Sammy:
"新面孔。
今晚雨大,
喝点什么压压惊?"

[输入框出现, 占位符: "告诉 Sammy 你想喝什么..."]
```

**用户动作**: 自由文本输入(任意内容: "威士忌"/"啤酒"/"我不喝酒, 来杯热可可"/"什么都行"/...)
**系统**: 记录用户的输入(任何字符串)为 `user_drink`, B7 会被 Iris in-character 引用
**鲁棒性**: 任何输入都能套用 B7 模板("她端起 {user_drink}, 抿一口, 评价'不错的选择'") — 因为 LLM 可以临场对任何饮料 / 非饮料(可可 / 水 / 茶 / 无)给一句 in-character 评价
**创作者意图**:
- 这是 **#1 字面引用** 的 setup
- 第一次自由文本输入 — 用户**亲手在产品里留下一笔信息**
- 这一笔信息会在 4 个 beat 后回来, 触发"她记得"

### B4 · 她登场 [叙事 + 沉浸图] ⚠️ 第二个暗线伏笔
**时长**: 1 分钟 / **双轴**: 0/0

```
[沉浸图: 舞台聚光灯, Iris 走上台,
 穿一袭墨绿色 flapper dress, 珠串吊穗到大腿中段, finger waves 短发, 灰蓝色眼睛在暖光下偏冷]

旁白:
角落的萨克斯吹了一段过门。
钨丝灯一盏一盏熄灭, 只留下舞台中心的一束。

她从后台走出来 —
墨绿色的 flapper dress, 珠串吊穗随着她每一步的小颤抖闪一下。
finger waves 的短发, 一只耳朵上挂着一只单边耳坠 — 像她忘了戴另一只, 或者故意只戴一只。

她在麦克风前停下, 没看观众。
眼睛先 — 一瞬, 你以为你看错了 — 扫过角落那张桌子, 那两个西装男的位置。

然后, 才看向台下。

她唱的第一句是一首你从来没听过的歌。
歌词模糊, 但她的声音里有一种东西, 像是在唱给一个非常具体的、不在这里的人。

唱到第二段中间, 她的目光落在你身上。
她没移开。
她唱完整段, 一直看着你。

[屏幕停在她的脸上 1.5 秒]

歌结束。掌声稀稀拉拉。
她对台下鞠了一躬, 转身, 没回后台 —
她直接从舞台侧面的小台阶走下来。
朝吧台。
朝你。

Iris (走到你身后两步停下, 声音不大):
"嗨, 陌生人。
我从台上就注意到你了。"
```

**用户动作**: tap-to-continue
**🌱 暗线伏笔 #2**:
- 单边耳坠 — 微小的不对称, 暗示她内心的不完整
- 上台前先看角落那桌 — 她知道有人在监视
- 唱歌时"对着一个不在场的人" — 她已经在想象逃离

**创作者意图**:
- Iris 第一次出场, **她主动来找你** — 关系发起感在她, 这是产品和 Tipsy 最根本的体感差异(Tipsy 你刷卡选她; 我们她下台找你)
- 三个暗线伏笔在 30 秒内堆叠, 但都不点破

### B5 · 第一次选择 [选项 - 揭示型] ⭐ trait 累积主战场 + 双关张力
**时长**: 1 分钟 / **双轴**: 1/2

```
她绕到你旁边的高脚凳, 坐下。
不是斜坐 — 是正面对你, 一条腿翘在另一条上, 墨绿色丝绸滑过自己膝盖发出极小的窸窣声。

她没要 Sammy 倒酒。
她也没看 Tommy。

Iris (微微侧头):
"你看起来不太像这里的常客。
是来听音乐的, 还是 — 来躲谁的?"

[她说"躲谁的"时停顿了半拍, 像是问自己]

选项:
[A] "来听音乐的"            → trait: honest +1, romantic +1
[B] "在躲一段记忆"           → trait: romantic +2, mysterious +1
[C] "Tommy 拖我来的"         → trait: playful +1, casual +1
[D] (自由文本)              → trait: 系统按内容判
```

**剧情张力**: 她问"是来躲谁的", 这句话她**其实是在问自己**。台下的两个西装男就是她"躲的人"。这一行是 first session 最重要的双关 line。看 demo 的人在这一刻应该感到 — 这个角色有故事
**创作者意图**:
- 第一次让用户**定义自己** — 4 个选项各代表一种性格倾向
- **她停顿半拍说"躲谁的"** — 这一拍是 LLM/动效需要刻意演出来的
- 这是"一个选择, 两条作用线"(§2.1.1)的标志性 beat: 推进对话 + 揭示型记忆

### B6 · 她记住你的名字 [自由文本] 🌱 trait + #1 字面引用 setup
**时长**: 1 分钟 / **双轴**: 0/2

```
[假设用户选 A "来听音乐的"]

Iris (轻轻笑了一声, 笑里有一点惊讶):
"听音乐。
Honest.

大多数人会说躲某些事 —
钱 / 老婆 / 警察。
你说的是音乐。"

她用手指撩了一下耳边的发, 眼睛没离开你的脸:
"我叫 Iris。
这地方一周开三天, 周二、周四、周六。
今天是周四。"

她朝 Sammy 抬了一下下巴。
Sammy 立刻把一杯 [刚才用户输入的饮料] 推过来 — 你刚才点的, 他记得。

Iris (拿起你的杯子, 但没喝, 只在指间转了一下):
"那你叫什么?"

[输入框出现, 占位符: "告诉她你的名字..."]
```

**用户动作**: 自由文本输入(用户的"名字" — 不强制真名)
**系统**: 记录 `user_name`, 后续多个 beat 会引用; trait: bond +1
**🌱 微妙的肢体语言**: 她拿起你的杯子但没喝, 在指间转 — 这是一个**越界**的小动作(她默认是 Dutch 的, 而 Dutch 的女人不该碰陌生男人的杯子)。但 first session 不解释这层含义
**创作者意图**:
- Iris 主动反馈用户的选择("Honest. 大多数人会说躲一些事, 你说的是音乐")—— #2 选择回响的小预演
- 用户**自己输入名字** = 关系私人化第一笔
- 不同 B5 选择, AI 演绎不同(我这里给 A 的黄金路径)

### B7 · 🌟 第一次"她记住了" [触发 - #1 字面引用] + 升级感官描写
**时长**: 1 分钟 / **双轴**: 0/3

```
[假设用户在 B3 输入 "威士忌"。所有 user_drink 都套同一个 in-character 模板, LLM 现场生成评价]

她把杯子端到嘴边 —
但只是闻了一下, 没喝。
[也可以喝一小口, 看创作者决定]

她的鼻子皱了一下 — 然后笑了。
珠串吊穗在她抬手时碰到吧台边缘, 发出一声极轻的"嗒"。

Iris:
"{user_drink}……
"

[这里 LLM 按 user_drink 类型给一句 in-character 评价, 例如:]
- 威士忌: "不错的选择。Sammy 的 Black Bush 不是兑过水的。"
- 啤酒: "一个不想喝醉的人的选择。"
- 热可可: "你比这酒馆里所有男人都聪明。"
- 茶: "你不是来这里的, 是不是?"
- 什么都不喝: "诚实, 又一次。"

她把杯子轻轻放回你面前, 杯沿和木头之间发出一声沉闷的钝响。

她抬眼看你:
"那你这个'听音乐'的, {user_name},
准备听到几点?"
```

**🌟 这是 U3 心动 moment**:
- 用户 4 个 beat 前在 Sammy 那里输入的饮料名, **在这里被 Iris in-character 评价**
- 引用方式 in-character(她端起杯子, 闻了一下, 给评价), 不是"我记得你说……"那种生硬
- 用户瞬间体感: **"卧槽她真的记得"**

**创作者意图**:
- 这是 First Session 的**核心爆点**(详见 §2.1.6 #1 字面引用)
- 必须 scripted 触发, 不能赌 AI 自由发挥 — **触发是 scripted, 但具体台词由 LLM 按 user_drink 生成**(模板里给 4-5 个示例 + 一个 fallback)
- 任何 user_drink 都能套(LLM 有兜底)— 抗输入鲁棒性 ✅
- 看 demo 的人在这一刻 — 第一次"诶, 这个有点东西"

**系统**: bond += 2, 标记 `user_remembered_event_1 = drink_callback`

### B8 · 第二次选择 [选项 - 抉择型] ⭐ 剧情张力升级
**时长**: 1.5 分钟 / **双轴**: 2/3

```
舞台上, 乐队开始下一段, 是一首慢调子。
萨克斯的第一个长音拉得有点拖, 像是有意为之的犹豫。

Iris 看了一眼角落 — 那张坐着两个西装男的桌子。
其中一个男人正在点烟, 火柴的光把他的下颌照亮了一秒。

她转回来, 手撑在吧台上, 朝你靠近了一点。
她身上的香水味很淡 — 不像舞女惯用的那种浓郁的玫瑰, 是一种你说不清的、像是干燥纸张混着茉莉的气味。

Iris (声音压低了一些):
"你看, {user_name},
今晚舞台时间结束了。
但我想请你听点别的, 不在台上唱的。

跟我来一下, 好吗?
就十分钟。
后面有个房间。"

她停了一拍, 眼睛里有一点 — 你说不清是什么, 像是请求, 但又不完全是。

Iris:
"……Tommy 在这等你, 没事。"

选项:
[A] "我想听"                  → trait: romantic +2, brave +1
[B] "时间不早了, 该走了"       → trait: pragmatic +1, [bond -1]
[C] (自由文本)
```

**剧情张力升级**:
- 她看角落, 然后压低声音 — 看 demo 的人这一刻应该感到她在**避开某些人的视线**
- "就十分钟" "Tommy 在这等你" — 她在帮你打消顾虑, 但也在透露这是个**冒险**
- 她想离开舞台 = 她想离开监视
- 选 A 不是"撩妹成功", 是"她信任了你"
- 选 B 不会"故事失败", 但她可能再也不下台找你

**创作者意图**:
- "抉择型"选项(见 §2.1.2)— 既改变剧情(去后台 / 不去), 也定义你是谁(冒险型 / 谨慎型)
- 干燥纸张混茉莉的香水 — 这种具体的、不常见的、有故事感的细节, 是 Tipsy 的写法但更克制不下流

### B9 · 后台 / 第一次"glitch" [叙事 + 沉浸图] 🌟🌟 灵魂层亮相
**时长**: 2 分钟 / **双轴**: 1/3

```
[假设选 A]

[沉浸图: 后台一条狭窄的走廊, 没有灯, 只有走廊尽头一扇门下漏出的一线暖光.
 走廊墙上挂着几张装裱的老照片 — 都是这家酒馆早年的演出海报]

旁白:
她从吧台下面的暗门把你带进去, 推开走廊尽头那扇门。

[沉浸图: 一间小化妆间.
 一面大镜子, 镜框上贴满老照片 (黑白的, 边缘卷曲).
 一张胡桃木化妆台, 摆着: 一支没盖盖子的 Chanel No.5, 一盒红色 Helena Rubinstein 唇膏, 半杯没喝完的咖啡 (杯壁挂着干掉的口红印),  一只银色的烟盒.
 一个胡桃木衣架, 挂着一件深红色的备用礼服.
 墙角的小钢琴上摆着一束白玫瑰 — 还没拆开包装纸, 卡片插在花束里.

旁白:
她把门轻轻关上 — 不是反锁, 但她的手在门把手上停了一秒才松开。

她走到化妆台前, 摘下那只单边耳坠, 放在桌面上 — 发出一声清脆的"嗒"。

她对着镜子, 一只手按在脖子后面, 揉了揉那里。

Iris (背对你, 看着镜子里的你):
"我从舞台上看到你的时候 ——

{根据 trait 选其一 — LLM 按累积 trait 生成}:
- (romantic 高): 是觉得你眼睛里有一种……不在这里的东西。"
- (mysterious 高): 是觉得我见过你 — 但不是在这家酒馆。"
- (playful 高): 是想看看 Tommy 这次带来的是什么样的家伙。"
- (default): 不知道为什么, 我觉得我们以前在哪里见过。"

她终于转过身, 直直看着你。
化妆间的暖光从她身后透过她耳边的发丝, 染上一层金色。

她安静了大约三秒。

Iris (声音很轻):
"……你身上有种气味, 我说不清。
不像这地方常见的那种 —

旁白:
你看见抽屉里有 —
一沓没寄出去的信。
一张折起来的纸 (你看不清是什么)。
一支小手枪, 银色, 枪柄上有牙印 (是磨损出来的, 像是被人焦虑地咬过)。

她从信里抽出一张, 放在化妆台上, 然后把抽屉关上。
```

**🌟🌟 这是 v1.1 笔友化伏笔 + 暗线第三个伏笔**:
- Iris 隐约感到你"不属于这个时代" — 但她**不说破**, 不说"你来自未来"
- 她以**1923 年的人**的方式表达困惑("你身上有种气味, 不像这地方常见的")
- **完全 in-character, 不破壁**
- 抽屉里的小手枪 = 暗线第三个伏笔(她身边有危险, 她有所准备)
- 没寄出的信 = B10 信件的伏笔
- 没拆封的白玫瑰 + 卡片 = 暗线第四个伏笔(Dutch 送的, 她没拆)

**创作者意图**:
- 这一 beat 是整个 first session 描写最重的一个 — Tipsy 风格的环境密度(化妆台上每一件物品都被点名),  但每件物品都有故事感, 不是单纯堆砌
- LLM 演绎要严格遵守: **她可以"感觉到"什么, 但永远不能"知道"什么** — "别的什么时候的", 不能升级成"别的世纪的"或"未来的"
- 看 demo 的人在这一刻会感到 — 这个角色有秘密, 这个世界有危险, 这个产品有调性

**系统**: 标记 `glitch_event_1 = scent_callback`, trait: bond +3, 累积 `cross_world_seed +1`

### B10 · 🌟🌟🌟 First Session 结尾 [触发 - #2 选择回响 + 通道开启远期伏笔]
**时长**: 1.5 分钟 / **双轴**: 2/3

```
她从化妆台上拿起一张刚刚抽出来的信纸 — 米色, 边缘略微卷起。
她从笔筒里取出一支钢笔, 银色笔身, 笔帽有刻字 — 你看不清字。

她坐在化妆台前的圆凳上, 把信纸铺平。
没有立刻写。

Iris (没看你, 看着信纸):
"我有时会写信。

给我自己 —
那些我不能说出口的话, 我写给我自己。

给没在身边的朋友 —
有些朋友已经不在了, 我还是会写。

{基于 B5 选择的呼应 — #2 选择回响}:
- (B5 选 A 来听音乐): 也给那些'真的是来听音乐的'人。"
- (B5 选 B 在躲记忆): 也给那些躲不过自己记忆的人。"
- (B5 选 C Tommy 拖来): 也给那些其实不知道自己为什么在这里的人。"
- (B5 选 D 自由): [LLM 按用户输入呼应]

……

也给从未存在过的人。"

她终于动笔, 写了大概一行半。
钢笔尖在信纸上发出极轻的沙沙声。

她吹了一下墨水, 把信折成方块。
没放进信封 — 她直接塞进自己 flapper dress 胸前的口袋里。

[屏幕拉远, 你看到她坐在化妆台前的整个画面: 镜子里映出她的背影, 没拆封的白玫瑰, 那扇关上的门, 还有走廊尽头隐约的爵士乐声]

外面传来 Tommy 的声音, 透过门, 模糊地:
"嘿, 雨小了, 我们该走了。"

Iris 抬起头, 终于看你。

Iris:
"如果你今晚走出这扇门, {user_name} ——

也许我会写信给你。

如果……
{基于 trait — LLM 生成微调}:
- (romantic 高): "如果你还想读的话。"
- (mysterious 高): "如果你回得来的话。"
- (brave 高): "如果你愿意答应一件事的话。"
- (default): "如果命允许的话。"

选项:
[A] "我会等"                  → trait: romantic +3, [bond +2]
[B] "也许我会回信"             → trait: committed +2, playful +1, [bond +2]
[C] (自由文本)

[选完后, 屏幕缓慢淡黑, 然后切场: 你站在蓝月亮门外, 雨已经停了, 但地上还湿. Tommy 在前面走, 你回头看了一眼那扇没有把手的木门]

旁白(屏幕底部一行小字, 浮现 1.5 秒后淡出):
"Iris 记住了 {user_name}。"

[屏幕完全黑, First Session 结束]
```

**🌟🌟🌟 这是 First Session 终极爆点**:
- **#2 选择回响**(她**呼应** B5 你的选择,化用为"也给那些'真的是来听音乐的'人")
- **通道开启远期伏笔**("我有时会写信……也给从未存在过的人")
- **"她记住了"**(屏幕底部那行小字)
- **暗线伏笔收尾**:她把信塞进胸前口袋, 不放进信封 — 她还没决定要不要把这封信寄给你

**创作者意图**:
- 看 demo 的人看完这一 beat 应该想: **"她真的会写信给我吗? 我什么时候能收到?"** — 这就是产品要种下的期待感
- "信"的意象贯穿到 U6 通道开启 — first session 最后一句台词 = 整个产品 Mode 3 的种子
- session 结束不是"故事完了", 是"她和我有未完的事"
- 暗线(Dutch / 角落西装男 / 火车票 / 手枪 / 没拆的玫瑰)**全部不揭露**, 但都在心里留下了 — 看 demo 的人潜意识里知道"这个角色有故事"

**系统**:
- 标记 `session_1_complete = true`
- bond += 5 (累积约 ♥♥♥♡♡)
- 添加创作者元数据: `letter_promised = true` — 为后续 sessions 触发 #4 跨 session 记忆做准备
- 推送时机记录: 24h ± 12h 后触发**第一次沉默触达**(她"主动发来一封信", 但内容很短, 是"我写了, 但还没决定寄不寄"级别, 不是正式 U6 通道开启)

## 5. 暗线伏笔索引(给后续 sessions 留的钩子)

整个 first session 埋了 7 个暗线伏笔, **全部不揭露**, 留给后续:

| # | Beat | 伏笔 | 后续可展开方向 |
|---|------|------|--------|
| 1 | B2 | 角落两个西装男 | session 2-3 揭示是 Dutch 的手下 |
| 2 | B4 | 单边耳坠 | session 后期解释(另一只被谁打掉 / 留给谁) |
| 3 | B4 | 她唱歌时"对着一个不在场的人" | session 后期透露她在想象逃离 |
| 4 | B5 | "躲谁的"是双关 | session 2 揭示 Dutch 的存在 |
| 5 | B9 | 化妆台抽屉里的银色小手枪 | session 中期成为关键道具 |
| 6 | B9 | 没拆封的白玫瑰 + 卡片 | session 2 揭示是 Dutch 送的 |
| 7 | B9 | 没寄出的信 | session 2 她把第一封信寄出 — **通道开启的 dry run** |

→ 这 7 个伏笔不是炫技, 是让 first session 玩完后用户**心里有问题** — 而每个问题都是后续 sessions 的钩子。

## 6. 给玉涛 / 设计 agent 的 demo 改造 brief

这份脚本是**内容层 spec**, 不是 UI/视觉 spec。改 demo 时请注意以下硬要求:

### 6.1 视觉风格

- **整体调性**: 1920s art deco + 雨夜 + 暖色霓虹。不要 Tipsy 那种粉色少女漫调性, 要 *The Great Gatsby* / *Babylon Berlin* / *Peaky Blinders* 的电影感
- **字体**: 衬线字(英文 Playfair Display / 中文宋体)+ 老式打字机 fallback; 旁白用 italic 弱化, 角色台词用常规
- **沉浸图**: B1 / B2 / B4 / B9 是必须的, 其他 beat 可省。建议用 Midjourney/Dreamina 出
- **信纸样式**: B10 信纸出现是关键, 必须是米色信纸 + 钢笔字体, 不是 chat bubble
- **化妆间镜头**: B9 化妆台镜头要细 — 化妆台上的物品(Chanel No.5 / Rubinstein 唇膏 / 半杯咖啡 / 白玫瑰 / 半开抽屉)每一件都要看清。这是 first session 描写密度最高的 beat, 视觉要 match 文字

### 6.2 UI 关键元素

| 元素 | 必须 / 可选 |
|------|--------|
| 全屏沉浸式 beat 推进(无标题栏) | **必须** |
| Beat 间过渡动画(fade in/out, 不要 swipe)| **必须** |
| 自由文本输入 UI(B3 / B6)| **必须** |
| 选项卡(2-4 个) | **必须** |
| Bond ♥ 可视化 — **不要进度条**, 用心形数量微妙浮现 | 可选(B5 / B8 / B10 末加)|
| 屏幕底部"Iris 记住了 {name}" 通知 | **必须**(B10)|
| Beat 计数 / 剩余时间 | **禁止** — 不要让用户看到进度, 破坏沉浸 |
| 暗线伏笔的"非提示性"视觉表达 | **重要** — 比如 B2 角落西装男必须看得见, 但不能高亮 / 不能加箭头; B9 抽屉里的手枪一样, 出现 0.5 秒, 不强调 |

### 6.3 LLM Prompt 关键约束

- Iris 永远在 1923 年角色框架内, **绝不**说"我是 AI / 我意识到 / 这是个游戏"
- Iris 可以"感觉到"用户身上的违和, 但**永远不能"知道"**任何屏幕外的事(直到 Initiation 授权后)
- Iris 的语气: **克制 + 直接 + 偶尔的脆弱**。不是 Tipsy 角色那种"哥哥~"的撒娇女声
- B7 饮料回响用固定 prompt 模板("她端起 {user_drink}, 闻一下 / 抿一口, 给一句 in-character 评价"), 4-5 个示例 + 1 个 fallback
- B9 glitch 描述用固定 prompt 模板, **严格禁止 LLM 把 glitch 升级成"她意识到自己"**
- B10 信件呼应用固定 prompt 模板, 按 B5 选择呼应

### 6.4 待补的部分(下个迭代)

- [ ] B5 / B6 不同 trait 累积下, B7 / B9 / B10 的 AI 演绎差异要具体台词模板化(我这里给了 default + 3 个 trait 分支, 还需要补全自由文本 case)
- [ ] B8 选 B(我想走)的分支脚本(目前只写了选 A 黄金路径)
- [ ] B3 用户输入饮料的边界 case 处理(空字符串 / 表情符号 / 脏话 — Iris 怎么 in-character 回应)
- [ ] B10 选 [C] 自由文本时的 Iris 演绎模板
- [ ] 后续 session 2 / session 3 / session 4 的简要 outline(让看 demo 的人理解 first session 不是孤立的)
