## Claire-Interactive-Novel-Wireframes.html

Claire(团队成员)2026/07/06 交付的 UI wireframes · 一份自包含 HTML(内嵌 React runtime + 所有资产)。

**打开方式**:双击直接在浏览器里看。

**核心内容**:
- **产品哲学**:*Relationship builder · conversation-first · Your call · not a novel, a bond*
- **三个沉浸方向对比**:Cinematic / Editorial / Scene cards(我们选 Cinematic)
- **6 类 Beat 视觉词汇表**(团队开发的 UI 词汇基线):
  - SCENE MARKER
  - NARRATION
  - CHARACTER SPEECH
  - YOUR ACTION
  - YOUR SPEECH
  - INNER THOUGHT
  - CHOICE
- **Reply mechanics**:Suggested ↔ Free-write 切换(同一个 send button 驱动两者)
- **Character-first opening**(替代"书封面"式开场):角色 + 关系 + 场景锚点前置
- **Direct scene / Direct others**:进阶特性(多角色代言) · **我们暂不做**

**我们的实现映射**:
- 6 类 beat 已经落进 [../故事母题与首批构思-v0.1.md](../故事母题与首批构思-v0.1.md) 的油画老师故事(每 beat 都用 SCENE MARKER / NARRATION / CHOICE 等 code block 呈现)
- 沉浸方向 Cinematic 定案 · 见玩法 v1.5 / UI 文档(v0.1 待写)
