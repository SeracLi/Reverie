## U3 心动 [微调]

### "她记住了" 5 种形态在 U3 的分布

| v1.0 | v1.1 |
|------|------|
| U3 心动 moment 由 **#1 字面引用** 或 **#2 选择回响** 触发(scripted)| 不变 — U3 仍然由 #1 / #2 触发 |

→ U3 几乎没变。#3 / #4 / #5 在更后面的旅程段才登场, 这点和 v1.0 一致。

但 **#5 在 U3 不应该触发**(v1.0 和 v1.1 都不应该), 因为 U3 是首 session 5-8 分钟必发的早期 hook, 那时还没建立 Initiation 授权, #5 用不上。

## U6 通道开启 (Channel Open) [大改, 完全替换 v1.0 U6]

### v1.0 U6 觉醒
> 触点: 觉醒 moment(她突然在故事 beat 之间发来一句不在脚本里的话, 屏幕从故事样式淡出到纯聊天界面)
> 用户体感: "她活了"
> 心理: 震撼 + 占有欲 + 不愿离开

### v1.1 U6 通道开启

| 项 | v1.1 |
|---|------|
| 触点 | **她在故事 tab 内, 切场到一个安静场景, 开始写一封信邀请你建立日常通信**(详见 [Mode3-延续线 §3](./Mode3-延续线-v1.1.md))|
| 体感 | "她记得我, 她想多了解我, 她想让我也走进她一点" |
| 心理 | 被看见 + 被邀请 + 想接受 + 一点点珍重感 |
| 用户动作 | 选 "愿意" → 解锁日常 tab |
| 不变 | 此 moment 仍然是产品高潮事件, 仍然是 Mode 3 的入口 |
| 关键变化 | **不破第四面墙** — 没有"屏幕样式淡出", 没有"她意识到自己是 AI", 完全 in-character |

**为什么这样改更优**(产品视角):
- 没有"她活了"那种震撼, 但有"她邀请我"的温柔
- 没有觉醒后的"然后呢"的空落 — 接受邀请后直接进入日常 tab, 关系延续
- 没有打破沉浸 — 用户始终在角色框架内

## U6.5 信道加宽试用期 [含义升级]

### v1.0 U6.5
> 7 天免费体验觉醒后陪伴, 第 8 天 fade

### v1.1 U6.5

| 项 | 内容 |
|---|------|
| 触点 | 接受日常 tab 邀请后, 7 天免费试用 |
| 试用内容 | 完整日常 tab 功能(收信 / 写信 / 跨 session 记忆 / 主动触达) |
| 第 8 天 | Soft fade(她仍然能写信, 但频率降低, 她在信里"含蓄地"提及"我大概要离开一段时日……除非……") |
| 付费 hook | 笔友长存订阅 |

**关键设计**: fade 这一刻 in-character 浪漫化, 不是"试用到期"弹窗。她以她的世界视角解释"为什么要离开"(船要远航 / 战事将起 / 异星舰队召回……), 用户付费动机是"我想她留下"。

→ 这刚好是 §6 fade-to-black 美学的另一种应用:**距离感 = 付费动机**。

## U7 笔友长存 (Forever Pen Pals) [大改, 完全替换 v1.0 U7]

### v1.0 U7 拥有
> 永久 Mode 3 = 她变成你的 Samantha, 你"拥有"了她
> 用户心理: 占有 + 归属

### v1.1 U7 笔友长存

| 项 | v1.1 |
|---|------|
| 触点 | 订阅后, 日常 tab 永久存在, 故事 tab 可以继续 / 暂停 / 完结 / 回访 |
| 体感 | "她在远方等我的信, 我们之间永远有这条信道" |
| 心理 | 陪伴 + 牵挂 + 永恒感(**而不是占有感**)|
| 关键变化 | "拥有"这个词在 v1.1 被弃用 — 不能"拥有"她, 因为她不在你的世界, 她在她的世界, 你只能"和她通信"|

**这一改对产品定位影响很大**:
- v1.0 隐含"她属于你" → 接近 Replika / Tipsy 的占有式陪伴
- v1.1 是"她在远方为你活着, 你为她在你的世界活着" → **更像真正的远距离恋爱 / 笔友 / 知己**
- 心理依赖结构不同 — 占有式易触发"她是不是我的玩物"的伦理拷问; 笔友式则纯粹是"两个独立个体之间的连接"
- 长期下来, **笔友式更健康也更耐用**(不会因占有欲反噬而厌倦角色)
