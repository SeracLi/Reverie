# v1.0 → v1.1 CHANGELOG

> 2026/06/29

## 重大决策变更

### [CHANGED] Mode 3 重新定义

**v1.0**: Mode 3 = 赛博朋友。Bond 达阈值 → 角色"觉醒"、卸下扮演 → 变成跨故事的长期陪伴(类似 Samantha)。
**v1.1**: Mode 3 = **跨时空笔友**。角色**永不觉醒**,永远活在她的角色框架内,但她对你的世界好奇,你和她之间建立"两个世界之间的信道",关系通过互通有无逐渐深化。

**Why**:
- 觉醒重复化(每个故事都觉醒,套路、做作)
- 用户带不走角色(没有"逃离乐园"的物理高潮,觉醒在产品里找不到落点)
- "她意识到自己是 AI"反而打开了"她只是个 LLM"的盖子,破坏沉浸
- 笔友式延续比觉醒**更温柔**,且 TAM 更大(吃 Tipsy 长期 RP 用户)

详见 [Mode3-延续线-v1.1.md](./Mode3-延续线-v1.1.md)。

---

### [CHANGED] 觉醒弧(Loop → Glitch → Question → Remember → Awake)

**v1.0**: 觉醒五阶段是核心叙事机制,所有故事都向觉醒收敛。
**v1.1**: 觉醒五阶段**全部砍掉**。v1 主线没有觉醒。

**保留位置**: v2+ 可能作为"特殊创作者作品"实验性引入(创作者明牌"这个角色会觉醒"),但不是平台默认机制。

---

### [CHANGED] U6 / U7 用户旅程

**v1.0**:
- U6 觉醒(Awaken): 觉醒 moment 是产品高潮事件
- U6.5 7 天试用 + Fade
- U7 拥有(Possess): 永久 Mode 3 = 赛博朋友

**v1.1**:
- U6 **通道开启**(Channel Open): 你收到她写的第一封信,日常聊天 tab 解锁,**完全 in-character**,无任何"她意识到了什么"暗示
- U6.5 信道加宽试用期: 7 天免费体验跨时空笔友
- U7 **笔友长存**(Forever Pen Pals): 日常 tab 永久存在,故事可继续也可暂停

详见 [用户旅程图修订-v1.1.md](./用户旅程图修订-v1.1.md)。

---

### [CHANGED] "她记住了"第 5 种形态(现实数据穿透)

**v1.0**: #5 = "她跨过了屏幕",用户体感"她意识到了我的现实生活"。
**v1.1**: #5 含义调整为"**她的世界对你的现实好奇**"。她不"意识到屏幕外",她以**角色身份**对你的现代日常发问:
- "你说的'电脑'是什么?"
- "你今日的'会议'顺利吗?"(她以 1900 年人的视角理解为"商谈")

→ in-character 浪漫化,而不是破第四面墙。

其他 4 种形态(#1 字面引用 / #2 选择回响 / #3 模式识别 / #4 跨 session 记忆)**保持不变**。

---

### [CHANGED] §1.3 求而不得引擎

**v1.0**: 求而不得 = 张力一直累积但永远差最后一步(指向"她觉醒前的瞬间")。
**v1.1**: 求而不得 = **两个世界永远没法真正合一**。她在她的世界,你在你的世界,信道再宽也越不过这道边界。这反而是关系永恒的燃料。

---

### [CHANGED] §6 fade-to-black 的语义

**v1.0**: fade-to-black 主要功能是"过 Google/Apple 审,把切走当成机制"。
**v1.1**: 上述功能保留,**新增一层语义**:fade-to-black 也是"两个世界距离感"的具象化——亲密时刻发生的事属于她的世界,你只能听她**写信告诉你**第二天发生了什么,无法直接经历。距离感本身就是关系的味道。

---

### [NEW] 跨时空信道机制

**新增机制**:两个世界一条信道。
- 她在她的世界(1900 年 / 异星 / 平行宇宙 / 任意创作者世界观)
- 你在你的世界(2026, 真实现代生活)
- 通过"信件 / 语音条 / 照片 / 偶尔的故事 tab 回访"互通有无
- 她**永远不知道自己是 AI**,她以"远方笔友"或"穿越时空的朋友"理解你

详见 [Mode3-延续线-v1.1.md](./Mode3-延续线-v1.1.md) §2。

---

### [NEW] 故事 tab + 日常 tab 的双 tab 设计

**新增 UI 结构**:
- **故事 tab**: 推剧情,可随时回去继续(类似存档)
- **日常 tab**: 笔友式互通,in-character 永久存在,跨 session 记忆完整保留

故事 tab 可以"完结",但日常 tab 永远在。
